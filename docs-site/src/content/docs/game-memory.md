记忆面板汇总了当前存档的目标设定、你手动钉住的关键事实、GM 自动积累的事实库、你的个人笔记，以及上一轮 GM 从知识库里检索到的参考资料。在这里你可以一眼看清 GM 当前"知道什么"，也可以主动写入或删除固定记忆和笔记。

入口：游戏右侧面板 → 顶部标签切换到「记忆」。

---

## 关键概念

### 主线任务与当前目标

面板最顶部展示两行文字：上方是来自剧本设定的**主线任务**，下方是 GM 在推进剧情中动态更新的**当前小目标**。这两项只读，不可在此修改。

### 固定记忆

标有图钉图标的条目列表。每一轮 GM 生成回复时，固定记忆里的内容**必然被注入上下文**，不会因为距离过远而被遗忘。适合放绝对不能遗漏的角色设定、关键约定、重要道具线索。

你可以手动添加或删除固定记忆条目。

### 事实库

由 GM 自动提取并写入，不支持手动编辑。每轮叙事结束后，系统从对话中抽取关键事实（NPC 状态变化、已发生的事件等）存入事实库，供后续轮次参考。

### 玩家笔记

供你自由记录的空间。笔记**不保证每轮都被 GM 读到**（不像固定记忆那样强制注入），适合放思路猜测、想法便签、对话摘录。你可以自由添加和删除笔记条目。

### 上轮检索

面板底部的区块，展示上一轮 GM 从知识库或历史对话里检索到的文本片段，以及本轮检索到的段落数（右上角数字）。让你了解 GM 当前参考了哪些背景资料。这一区块只读，每轮自动刷新。

## 常见任务

### 添加固定记忆

在「固定记忆」标题右侧点击 + 按钮 → 在弹出的输入框里输入文字 → 确认。该条目立即出现在列表中，下一轮 GM 就会读到它。

### 删除固定记忆

找到想删除的条目 → 点击条目右侧的关闭按钮 → 弹出确认提示后确认。删除后该条目不再被注入上下文。

### 添加玩家笔记

在「玩家笔记」标题右侧点击 + 按钮 → 输入内容 → 确认。笔记不影响 GM 的必读列表，只是你的私人备忘。

### 删除玩家笔记

找到想删除的笔记条目 → 点击条目右侧的关闭按钮 → 确认弹窗。

### 查看本轮 GM 参考了什么

直接看面板底部的「上轮检索」区块，里面是原文或历史对话的片段，以及本轮检索到的段落数。

## 常见问题

**固定记忆和玩家笔记有什么区别？**
固定记忆每轮必进 GM 上下文（有 token 配额，请勿堆砌无关内容）；玩家笔记只供你参考，GM 不保证每轮都读到。

**固定记忆放多少合适？**
没有硬性上限，但每轮注入有 token 预算。固定记忆过多时，优先级低的条目在高压轮次里可能被压缩。建议只放真正关键的设定，其余放玩家笔记。

**事实库的内容我能删除或修改吗？**
不能。事实库由系统后台自动维护，面板里只读展示，不支持手动编辑或删除。

**上轮检索是什么时候更新的？**
每一轮 GM 完成叙事后自动更新，展示本轮从知识库或历史对话里检索到的参考片段。刷新面板即可看到最新内容。

**这里的修改会影响其他存档吗？**
不会。记忆面板操作的是当前这局游戏的运行时状态，修改只影响本存档。
