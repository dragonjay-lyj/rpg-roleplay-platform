「剧本复核」用于检查 AI 从剧本原文中自动提取的设定是否准确。提取内容包括规范实体（人物、地点、阵营等）和规范世界线；你可以在这里查看 AI 生成的摘要、修改有误的条目、删除误识别的实体，并在确认无误后点击「确认设定无误」完成核对。

核对通过是剧本发布（公开分享）的前置条件；开局不强制要求，但建议核对后再游玩，以确保 GM 使用的设定准确。

入口：剧本管理 → 展开某剧本 → 点击「设定核对」按钮；或在操作下拉菜单中选择「设定核对」。

---

## 页面内容

### 核对状态栏

页面顶部显示当前核对状态：

- 橙色边框「此剧本的设定还没核对」——尚未通过核对，点「确认设定无误」完成核对。
- 绿色边框「此剧本的设定已核对通过」——已核对，同时显示核对时间。若提取结果有误，可点「撤回确认」回到未核对状态再编辑。

### 提取质量标记

核对状态栏下方展示若干提取质量信息：

| 标记 | 含义 |
|---|---|
| 提取正常 / 需核对 | AI 整体判断提取是否存在可疑问题 |
| 作者非正文 N | 检测到 N 处作者附注（夹注、括号说明等非正文内容） |
| 怪标题 N | 检测到 N 个格式异常的章节标题 |
| 编号缺口 N | 章节编号中存在 N 处跳号，可能有章节遗漏 |
| 广告清洗 N 行 | 导入时自动清洗掉的广告行数 |

这些标记帮助你快速定位需要重点检查的地方，本身不影响核对流程。

### 规范实体表格

列出 AI 从全书提取的实体，每行包含：

| 列 | 说明 |
|---|---|
| 名称 | 实体在原文中的规范名称 |
| 类型 | 实体分类（如 character、location 等） |
| 首现章 | 该实体在哪一章首次出现 |
| 重要度 | AI 评估的重要程度 |
| 摘要 | AI 生成的描述，可手动修改 |
| 操作 | 改摘要 / 删 |

### 规范世界线列表

页面下方展示该剧本的规范世界线，以及每条世界线下的章节节点序列。主时间线以星标标出。

## 常见任务

### 修正错误摘要

1. 找到摘要有误的实体行
2. 点「改摘要」，行内输入框出现
3. 修改内容后点「存」；如需放弃点「取消」

### 删除误识别实体

找到被错误提取的条目（如广告文字、章节标题被误识别为人名）→ 点「删」→ 确认删除弹窗。

删除不可撤销。若误删，需重新运行提取恢复，手动修改过的摘要会丢失。

### 完成核对

抽查无误后，点顶部「确认设定无误」按钮，状态栏变为绿色，核对时间随即记录。

### 撤回核对后重新编辑

若发现新的错误，点「撤回确认」将状态重置为未核对，编辑完成后再次点「确认设定无误」。

## 常见问题

**摘要一列是空的？**
该实体在原文中出现次数较少，AI 未能生成有效描述。可手动点「改摘要」补充。

**核对完后修改能立即生效吗？**
摘要的保存即时生效，下次游玩时 GM 会使用修正后的摘要。

**剧本更新后需要重新核对吗？**
是。重新运行提取（如触发重建）后，核对状态会自动重置为「未核对」，需重新进入本页核对确认。

**分享剧本时提示「分享前需先核对剧本设定」？**
这是发布闸：剧本必须先通过设定核对才能公开分享。完成核对后回到剧本列表再点「分享/公开」即可。

**章节切分有问题，在这里能改吗？**
不能，章节层面的操作（重命名、拆分、合并、重切）在「查看章节」弹窗里进行，见[剧本管理](/scripts)。
