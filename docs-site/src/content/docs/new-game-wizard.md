「新建游戏向导」引导你完成从选剧本到配置出身的全部建档工作。填完必填项后点「**创建并进入**」即可直接进入游戏；存档草稿会自动本地保存，中途关闭弹窗或切页再回来内容不会丢失。

入口：存档管理页右上角「**开始新游戏**」按钮。

---

## 向导区块说明

向导以滚动表单的形式展示各个区块，右侧「**概要**」栏实时汇总当前已填内容。

### 基本信息（必填）

- **存档名称**（必填）：给存档起个名字，可在创建后于存档详情页重命名
- **剧本**（必填）：从已导入的剧本中选择；若列表为空，需先去「剧本管理」导入剧本

若剧本仍在后台导入或缺少必要数据（章节/锚点），会显示提示且无法选择，需等导入完成再重试。

### 选择角色（必填）

角色来源分三个选项卡：

- **使用现有**：从账户下的玩家身份与用户角色卡中选择，支持预览
- **本剧本 NPC**：直接扮演当前剧本里的原著角色——选定后出身会自动锁定为「本世界人」
- **新建角色卡**：填写角色名称（必填）、身份定位、背景经历，创建后自动加入角色卡库

### 出生点（必填）

选择故事开局的时间锚点。先选好剧本后，出生点列表才会加载。列表按剧情阶段分组，展开对应阶段后可选择具体锚点（含章节范围和剧情摘要提示），不选则默认从故事开头开始。

### 角色出身与身份（第 2–4 步）

此区块分三个子步骤：

**第一步：出身来源**

决定你以何种方式进入故事世界，影响 GM 对你记忆、知识和灵魂设定的处理：

| 出身 | 含义 | 身份卡约束 |
|---|---|---|
| **灵魂穿越** | 外来灵魂住进本地肉身，保有现代记忆与先知 | 可挂任意来源的身份卡，也可不挂 |
| **整体穿越** | 肉体与灵魂整体穿越，彻底外来者 | 无本地身份，只能选「不挂身份卡」 |
| **双魂同体** | 与原住民灵魂共处同一肉体，GM 扮演另一魂 | 必须挂一张身份卡（不可不挂） |
| **本世界人** | 你就是这个世界的原住民本人 | 可不挂，或补充一个社会定位 |

选「本剧本 NPC」角色时，出身自动锁定为「本世界人」，不可手动更改。

**第二步：角色身份（可选）**

身份卡是叠加在角色卡之上的定位 overlay（社会身份、代号、关系网络），与角色的姓名外貌无关。共四种来源：

- **不挂身份卡**：直接用所选角色卡开局（整体穿越专用路径；其余出身也可选此项）
- **从原著角色**：从剧本 NPC 卡中选一位作为主角遗忘或占有的真实身份
- **AI 生成**：点「生成身份」，根据剧本、出生点和所选角色生成 4 个候选，点选其中一个；不满意可点「重新生成」
- **手动填写**：填写「别名」「定位」「背景」后点「确认身份」

**第三步：是否知道身份（仅挂了身份卡且出身非整体穿越时有效）**

- **知道**：开局即清楚身份的社会关系与秘密
- **不知道**：失忆/初醒状态，身份在剧情中逐步揭开

### 引导与防剧透（可选）

| 设置 | 选项 | 说明 |
|---|---|---|
| **元知识/先知** | 无先知 / 部分 / 全知 | 你作为玩家对原著剧情的了解程度 |
| **NPC 起疑阈值** | 迟钝 / 多疑 | 世界中的 NPC 对你异常行为的敏感度 |
| **故事引导** | 强引导 / 适度 / 自由 | GM 推动故事走原著轨道的力度 |
| **防剧透等级** | 严格 / 宽松 | GM 保护原著剧情不被提前揭露的力度 |

以上设置均有默认值，跳过即生效；进入游戏后也可在设置面板调整（引导强度在游戏内实时可改）。

### 高优先级 GM 引导（可选）

用自然语言告诉 GM 你的游戏方向与偏好，例如「主角拒绝任何战斗，必须找非战斗解决方案」或「穿越者身份是绝对秘密」。这段文字会以高优先级注入每轮上下文，GM 须尽量遵守。此项可留空。

## 常见任务

### 最简流程（4 步进入游戏）

1. 在「基本信息」填写存档名称并选择剧本
2. 在「选择角色」选一张现有角色卡（或新建）
3. 在「出生点」展开一个阶段，选中一个锚点
4. 点「**创建并进入**」

### 扮演本剧本 NPC

1. 在「选择角色」切到「**本剧本 NPC**」选项卡
2. 选一位原著角色
3. 出身会自动锁定为「本世界人」，GM 将严格守这个角色的世界观

### 用 AI 生成身份卡

1. 先选好剧本、出生点和角色
2. 在「角色身份」区块选「**AI 生成**」
3. 点「**生成身份**」，等待 4 个候选出现
4. 点选其中一个；不满意可点「**重新生成**」再试

### 失忆开局

1. 挂上一张身份卡（AI 生成或从原著角色选）
2. 在「是否知道身份」选「**不知道**」
3. 游戏开始时角色对自己的真实身份一无所知，随剧情逐步揭开

## 常见问题

**剧本下拉为空或显示「先导入一部剧本」？**
先去「剧本管理」页面完成至少一部剧本的导入，处理完毕后回来重新打开向导即可。

**出生点列表加载不出来？**
先在「基本信息」区块选好剧本，出生点才会加载。若剧本刚导入完成，可点「重试」刷新。

**「本剧本 NPC」选项卡是灰色、不可点？**
该剧本还没有提取到 NPC 角色卡。可在剧本编辑器里手动生成或提取人物卡后再试。

**双魂同体为什么必须挂身份卡？**
双魂同体需要一个原住民本体让 GM 扮演另一魂；没有身份卡，GM 无从知晓这具身体的来历与关系，叙事无法成立。

**选错出身能改吗？**
出身在建档时确定后便锁死，游戏开始后无法修改。如需更换出身，请新建存档。

**向导填到一半关掉，下次还在吗？**
向导内容会实时写入本地草稿，重新打开时自动恢复上次的填写进度。

**存档名称之后能改吗？**
可以，进入「存档管理」选中该存档，在详情面板找「重命名」操作即可。
