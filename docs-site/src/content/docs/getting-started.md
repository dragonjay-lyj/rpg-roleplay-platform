本文带你从零走完 RPG Roleplay 的第一次完整上手——注册账号、填入 LLM API Key、导入小说、新建存档并开始游玩。全程约 10–15 分钟，不含剧本提取等待时间。

---

## 第一步：创建账号 / 登录

打开 RPG Roleplay 后，若尚未登录会自动跳转到登录页。

**注册新账号**

1. 在登录页切换到「注册」标签。
2. 填写用户名、密码、显示名称和邮箱。
3. 勾选《服务条款》《隐私政策》《可接受使用政策》及年龄确认，点「注册」。
4. 系统会向你填写的邮箱发送验证码，输入后完成注册。

**已有账号直接登录**

在登录页填写用户名和密码，点「登录」即可。登录成功后跳转到主界面。

> 用户名不区分大小写，但 `@` 等特殊字符会被保留，请原样填写注册时使用的用户名。

---

## 第二步：添加 LLM API Key

RPG Roleplay 是 BYOK（Bring Your Own Key）模式——平台不内置共享密钥，你需要填入自己从各 AI 供应商申请的 API Key，游戏 GM、提取、生图等功能才能工作。**未添加任何 Key 时，模型选择器为空，无法开始游戏。**

入口：顶栏「设置」→「模型配置」（详细说明见[模型配置](/settings-models)）。

1. 点击页面右上角「添加 API Key」按钮。
2. 在弹窗的「供应商」下拉中选择你拥有 Key 的服务商（如 OpenAI、Anthropic、DeepSeek、Google AI Studio 等）。选择后 Base URL 会自动填入默认值。
3. 在「API Key」框中粘贴 Key，点「添加」。
4. 平台自动向供应商同步可用模型列表，该供应商出现在配置列表中，状态变为绿色可达后即可使用。

常见供应商申请地址：

| 供应商 | 申请地址 |
|---|---|
| OpenAI | platform.openai.com |
| Anthropic | console.anthropic.com |
| Google AI Studio | aistudio.google.com |
| DeepSeek | platform.deepseek.com |
| OpenRouter（聚合） | openrouter.ai |

> **国内直连问题**：海外供应商（OpenAI / Anthropic / Google）在国内可能无法直连。添加 Key 时在「连接方式」选「HTTP 代理」并填写本地代理地址（如 `http://127.0.0.1:7890`），或使用 OpenRouter 这类聚合代理作为替代。详见[模型配置](/settings-models)中的「常见问题」。如果添加 Key 后仍无法正常使用，请访问[故障排查](/troubleshooting)。

## 第三步：导入小说剧本

RPG Roleplay 以「剧本」为核心，每一局游戏都基于一本已导入的小说展开。

入口：顶部导航栏「剧本」→ 右上角「导入剧本」（详细说明见[剧本管理](/scripts)）。

1. 点「导入剧本」，进入导入页面。
2. 拖入或点击选择 `.txt` 文件（UTF-8 编码），推荐文件大小在 50MB 以内。
3. 系统弹出「章节切分预览」，显示识别到的章节数、总字数和置信度。置信度高于 85% 显示绿色，表示自动识别准确；若偏低可点「换规则重试」手动指定切分规则。
4. 确认章节预览无误后，点「确认导入」。

**导入后台会依次完成以下工作**（需要 LLM Key，耗时因剧本长度而异）：

- 章节切分（秒级）
- NPC 角色卡生成
- 世界书条目提取
- 时间线锚点构建
- 向量索引构建

可在「剧本管理」的列表「状态」列查看就绪进度。**剧本各模块就绪前，新建向导中该剧本会标注「未就绪」并禁止选择，请等待提取完成再继续下一步。**

> 如果希望只切分章节、跳过 LLM 提取阶段，可在导入时关闭「生成角色卡」和「生成世界书」开关，之后在剧本详情面板按需手动触发。

## 第四步：新建存档并开始游戏

入口：顶部导航栏「存档」→ 右上角「新游戏」（详细说明见[新建向导](/wizard)和[存档管理](/saves)）。

**向导分四个必填区块：**

1. **基本信息**：填写存档名称（可用默认预填），在「剧本」下拉中选择已就绪的剧本。若剧本显示「未就绪」，说明上一步的提取尚未完成，请等待后再试。

2. **选择角色**：选择你要扮演的角色卡。
   - **我的卡库**：使用已有角色卡或 Persona
   - **本剧本 NPC**：直接扮演原著角色（出身自动锁为「本世界人」）
   - **新建角色卡**：填写姓名等字段立即创建

3. **出生点**：从列表中选择故事开局的时间锚点。锚点按剧情阶段分组，每个锚点旁有章节范围和摘要说明。选好剧本后列表才会加载。

4. **出身与身份卡**（可选）：选择你进入故事世界的方式（魂穿 / 肉穿 / 双魂同体 / 本世界人），以及是否叠加一个身份卡。跳过直接使用默认值也可以。

右侧概要栏实时显示已填项目和完成状态，全部必填项打勾后「开始游戏」按钮变为可点击。点击后存档创建完成并直接进入游戏。

---

## 第五步：进行第一回合

进入游戏后，界面底部固定显示输入面板（详细说明见[输入面板](/input)）。

**发送你的第一条消息：**

1. 在底部输入框中输入行动、对话或场景描述。
2. 按回车发送（默认），或点右下角「发送」按钮。GM（AI 游戏主持人）会以流式方式输出回应。

**几个常用操作：**

- 输入 `/` 弹出斜杠命令菜单（更新位置、时间，固定记忆等）
- 点「继续」让 GM 主动推进剧情，无需自己输入内容
- 点右侧模型标签切换当前使用的 AI 模型
- GM 输出时点「停止」可随时中断生成

**第一次接触 GM 的建议**：直接用自然语言描述你的角色正在做什么，或说一句台词，GM 会根据剧本世界观接续叙事。不需要记住任何命令格式即可开始。

## 常见首次上手问题

**新建向导里剧本下拉为空**
需先完成第三步的剧本导入，且等待章节切分与锚点构建完成（状态列显示就绪）后才可选择。

**剧本显示「未就绪」，无法选择**
剧本正在后台提取，耗时取决于剧本长度和 LLM 响应速度，一般几分钟到十几分钟不等。可在「剧本管理」页面查看进度，等待状态变绿后重新打开向导。

**游戏内模型选择器是空的**
尚未添加任何 LLM API Key，或已添加 Key 但供应商状态为「不可达」。回到「设置 → 模型配置」检查 Key 和网络连通性。详见[故障排查](/troubleshooting)。

**按回车发送但我想换行**
点击输入框右下角「↵」切换按钮，关闭「回车发送」模式，之后回车只换行，Cmd/Ctrl + 回车才发送。
