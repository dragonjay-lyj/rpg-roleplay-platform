「世界书」是 GM 的背景知识库——记录剧本世界中的地名、人物关系、势力、规则约束等设定。导入剧本后，系统会自动从原著章节内容提取世界书条目；你也可以在此手动添加、编辑或整理。

每轮 GM 回复前，系统会按相关性从世界书中选取若干条目注入上下文，让 GM「知道」这个世界的规矩、地理和人物关系。启用了「世界知识」记忆桶后，注入行为才会生效；可在「记忆设置」中调整 token 上限。

入口：剧本管理 → 选中剧本 → 详情面板「**世界书**」标签。

## 关键概念

### 条目结构

每条世界书条目包含以下字段：

- **标题（title）**：条目名称，在列表中显示，也用于语义匹配
- **内容（content）**：具体设定文字，注入时 GM 读到的正文
- **优先级（priority）**：数字 0–1000，越高越优先被选中注入；默认 50。表格默认按优先级降序排列
- **启用（enabled）**：关闭后该条目完全不参与注入，等同于临时屏蔽；不会被删除
- **标签（tags）**：附加在条目上的关键词组，辅助语义匹配

系统会根据优先级给条目打一个子类型标签，方便快速识别重要程度：
- priority ≥ 80 → core
- priority ≥ 50 → major
- priority ≥ 20 → minor
- 其余 → detail

### 三路注入机制

系统并非每轮全量注入所有条目，而是综合三路机制动态选取：

1. **常驻注入**：优先级极高的条目（如世界核心规则）每轮必然出现
2. **关键词/标签匹配**：条目的 tags 中含有当前对话上下文出现的关键词，优先触发
3. **RAG 语义召回**：系统对当前场景做向量检索，召回语义最相关的条目

三路同时参与，合并去重后按 token 预算截断，注入到 GM 上下文。

### enabled 闸

`enabled` 是独立于优先级的开关。禁用一条世界书条目后，无论其优先级多高，该条目都不会被注入，但条目本身保留在列表中，随时可重新启用。适合临时屏蔽剧透设定。

### 来源与防重建覆盖

每条条目都有一个来源字段（`source`）：
- **editor**：由用户或 AI 手动创建/编辑。重建知识库时此类条目受豁免，**不会被覆盖或复活**，删除即永久删除。
- **extracted / 其他**：从原著自动提取。重建知识库时这些条目会从原著重新生成。删除后若触发重建，条目会重新出现——批量删除时系统会提示你受影响的条目数量。

### 防剧透：章节门控

系统后端支持按当前游戏进度决定哪些条目可见。尚未到达的章节对应的剧透设定，不会注入到当前 GM 上下文中，避免 GM 提前透露情节。这一机制在后台自动运作，无需手动配置。

## 常见任务

### 查看与搜索条目

剧本管理 → 选中剧本 → 详情面板「**世界书**」标签，列出该剧本所有世界书条目。

- 顶部搜索框支持按标题或内容关键词过滤
- 点击列表标题（标题/优先级/启用）可切换排序
- 每页最多显示 50 条，超出时出现翻页控件

### 新建条目

点击右上角「新建」按钮，在右侧抽屉中填写标题、内容、优先级和标签，保存后立即生效。手动新建的条目来源标记为 editor，重建时不受影响。

### 编辑单条条目

点击任意行，或点击行右侧「详情」按钮，右侧打开编辑抽屉，可修改全部字段。修改后点「保存」，放弃修改点「撤销」。

### 快速调整优先级或启停状态

直接在表格列上点击可以快速修改标题或优先级（inline 编辑），无需打开抽屉。

### 批量操作

勾选多条条目后，列表上方出现批量操作工具条：

- **批量启用 / 批量禁用**：一键将所选条目的 enabled 设为开/关，单次事务提交，不需要逐条修改
- **批量设优先级**：在数字输入框填入目标优先级（0–1000），点「设置优先级」，所选条目统一更新
- **批量删除**：物理删除所选条目，操作前会弹出确认对话框；若所选中含有来源非 editor 的条目（即自动提取的条目），系统会额外提示这些条目在重建知识库后可能重新出现

### 游戏中查看实时世界书状态

进入游戏 → 右侧面板「**世界书**」标签，显示当前场景的位置、时刻、天气、故事阶段，以及世界规则约束列表和已触发的关键词。这些字段可在此处直接点击编辑（即时生效）。

## 常见问题

**条目太多，GM 会不会乱？**
不会。系统按 token 预算选取最相关的条目，不会全量注入。可在「记忆设置」的「token 上限」中调整注入容量，并用「世界知识」记忆桶开关整体控制是否注入。

**怎么确保某个重要设定每轮都被 GM 看到？**
把该条目的优先级设为 80 以上（core 级别），会优先进入常驻注入池。如果需要铁定每轮必有，可考虑把内容加入「固定记忆」——固定记忆优先级高于世界书。

**禁用条目和删除条目有什么区别？**
禁用只是暂时屏蔽，条目保留，随时可重新启用。删除是物理删除，无法撤销；如果条目来源是自动提取（非 editor），下次重建知识库时还会重新出现。

**我手动删了某条自动提取的条目，为什么又回来了？**
这是正常行为：重建知识库时，原著自动提取的条目会从正文重新生成。如果你不希望某条自动提取的内容出现，请禁用它（关闭 enabled），而不是删除。

**剧本不是我创建的，能编辑世界书吗？**
只有剧本所有者可以编辑。如果你访问他人剧本的世界书，界面会显示只读提示，并提供「另存为副本」按钮，fork 后可在自己的副本上自由编辑。
