游戏主界面底部的输入区是你与 GM（游戏主持人）对话的核心入口，除了普通对话，它还支持斜杠命令、附件、角色提及和权限控制。

入口：在任意游戏存档界面，底部工具栏即是输入区。

---

## 普通对话

在输入框里直接写你想做的事、说的话、或给出行动描述，然后发送。GM 会读取你的输入，结合当前剧情状态给出叙事回应。

- 输入框支持自动撑高，最多展开到 280px 高度
- 发送后 GM 流式输出，可随时点「停止」中断
- 支持将文本直接拖拽到输入框插入

如果当前场景下玩家角色处于被动状态（昏迷、旁观、过场），点击工具栏里的「继续」按钮可直接让 GM 推进一段剧情，无需输入任何文字。

---

## 发送方式切换

工具栏右下角显示当前发送快捷键，旁边有一个切换按钮：

- **Enter 发送模式**（默认）：直接按 Enter 发送，Shift + Enter 在输入框内换行
- **Cmd/Ctrl + Enter 发送模式**：Enter 换行，Cmd + Enter（Mac）或 Ctrl + Enter（Windows）才发送

点击切换按钮可在两种模式间切换，设置自动保存到本地，刷新后仍然有效。IME 输入法确认阶段按 Enter 不会误发。

## / 命令

在输入框里输入 `/` 会弹出命令选择菜单，用 `↑↓` 方向键选择、Enter 确认、Esc 取消。也可以直接打出命令全名后跟参数。选中命令后输入框顶部会显示命令标签，按 Backspace 可移除。

### 查询类

| 命令 | 说明 |
|---|---|
| `/status` | 显示当前状态摘要（角色、位置、时间等） |
| `/debug` | 查看上一轮 GM 的内部检索结果，帮助排查问题 |

### 状态写入类

| 命令 | 示例 |
|---|---|
| `/set 自然语言描述` | `/set time=dawn; location=harbor` — 用自然语言批量强制改一组游戏参数，写入即落盘，优先级高于 GM 自动派生 |
| `/loc 地点` | `/loc 雾港码头` — 快速更新所在位置 |
| `/time 时刻` | `/time 次日清晨` — 推进时间线 |
| `/rel 角色 关系` | `/rel 沈知微 同盟` — 更新人物关系 |
| `/var 变量=值` | `/var phase=港口黄昏` — 设置世界线自定义变量 |

### 记忆类

| 命令 | 说明 |
|---|---|
| `/pin 文本` | 把一段内容加入固定记忆，GM 每轮都会参考 |
| `/note 文本` | 玩家私人笔记，不影响 GM |

### 模式类

| 命令 | 说明 |
|---|---|
| `/memory normal\|deep\|off` | 切换记忆检索模式（normal = 默认；deep = 精细检索；off = 关闭） |
| `/permission default\|review\|read_only\|full_access` | 切换 GM 写入权限（见下方「权限控制」） |

### 工程类

| 命令 | 说明 |
|---|---|
| `/save` | 手动保存当前存档 |
| `/retry` | 重试 GM 上一轮输出（换个结果） |

## @ 提及角色

在输入框里输入 `@` 会弹出角色列表，列出当前存档中的关系人物以及玩家自身。选择后在输入框里插入 `@角色名`，GM 会把这个标注视为你在针对该角色发言或行动。用 `↑↓` 方向键导航，Enter 或 Tab 插入，Esc 关闭。

角色列表来自当前存档的关系数据，新存档第一轮前列表可能为空。

## 附件（+ 按钮）

点击工具栏左侧的 + 图标打开附件菜单，可以附加以下内容：

### 本地文件

- **文件 / 文本**（TXT、MD、JSON）：上传一段文字或文档，GM 会在本轮参考其中内容
- **图片**（PNG、JPG、WEBP）：上传图片，GM 支持多模态理解，可以描述、分析图中内容

### 剧本资源

- **插入章节**：从当前剧本目录搜索一个章节，注入为上下文
- **角色卡**：搜索或拖入一张角色卡，让 GM 参考该角色的详细设定
- **世界书条目**：全文搜索世界书，把匹配条目注入 GM 的上下文

### 能力扩展

- **MCP 工具**：挂载一个已配置的 MCP 服务器工具，让 GM 在本次对话里调用外部能力（详见[MCP 是什么](/mcp)）
- **Skill 包**：挂载一个规则裁定包（如骰子规则 / 5E 规则集），让 GM 按照规则包进行裁定
- **计划模式**：预演一条叙事路径，但不写回存档状态

附件显示在输入框上方，点击附件右侧的 × 可移除单个附件。

## 权限控制

工具栏左侧显示当前 LLM 写入权限，点击可弹出选择菜单：

| 模式 | 说明 |
|---|---|
| **只读** | GM 所有状态写入都需要玩家手动审批，最保守 |
| **默认** | 白名单字段（位置、时间等）自动写入，其他需审批 |
| **自动审查** | 白名单加上关系 / 世界变量也自动写入，其他需审批 |
| **完全访问** | 除硬黑名单外全部自动写入 |

当 GM 提出写入申请或询问时，输入框上方会出现待确认条目。写入申请可展开查看变更前后的值和理由，然后逐条允许或拒绝；GM 问句会显示选项供你点选回答。如果 GM 检测到套路化比喻，也会在此区域提示你是否要重新生成本轮。

---

## 模型切换

工具栏右侧显示当前 GM 使用的 AI 模型，点击弹出模型选择浮层。浮层内列出所有已配置凭证的可用模型，可查看模型的健康状态和定价信息。切换只影响当前存档，不影响其他存档。

浮层底部还有 **Effort** 档位选择，控制该模型的思考深度：

| 档位 | 说明 |
|---|---|
| Off | 禁用思考，最快最省 |
| Low | 约 1k tokens 思考 |
| Medium | 约 4k tokens 思考 |
| High | 约 8k tokens 思考（默认） |
| Extra | 约 16k tokens 思考 |
| Max | 约 24k tokens 思考，最深推理，最贵 |

Effort 设置按模型分别保存，切换模型后会自动读取该模型上次的 Effort 设置。

---

## 上下文用量

工具栏右侧有一个圆环进度指示，显示当前上下文占用比例：超过 70% 变橙色，超过 90% 变红色。点击圆环会展开详细面板，显示各部分（记忆、世界书、对话历史等）的 Token 占用量和分布条。上下文接近满载时建议存档后开启新对话。
