# AI 生图

平台支持在多个场景下用 AI 生成图片：角色卡头像、人设图、剧本封面、聊天内插图、游戏场景图。所有生图功能均需自带 API Key（BYOK），平台本身不提供共享额度。

入口：根据场景不同，入口分别在「角色卡」编辑页、「剧本」编辑页、游戏/酒馆聊天框工具栏，以及「设置 → 模块模型」。

---

## 这页能干嘛

生图功能让你为故事中的角色、剧本封面、聊天场景配上 AI 生成的插图，与文字叙事配合使用。你可以写提示词描述想要的画面，选择生图模型，调整分辨率比例，生成后直接应用到对应位置。

**统一图片弹窗（MediaStudio）** 整合了三个图片来源：AI 生成、本地上传、从图库中选。凡是角色头像、剧本封面等位置点击更换图片，都会打开这个弹窗。

---

## 前置条件：配置生图模型

生图功能属于 BYOK 模式，必须先在「设置 → API 密钥」配置支持图像生成的服务商 API Key，再在「设置 → 模块模型 → 图像生成模型」选好对应的 provider 和模型，才能正常生成。

支持的服务商类型包括：Vertex AI（Gemini）、DashScope（通义万象）、豆包、以及 OpenAI 兼容的生图端点（含 OpenRouter 等中转站）。

未配置 Key 时点击「生成」会提示「尚未配置生图模型 API Key」并给出跳转链接；Key 填写有误（鉴权失败）则显示具体错误原文，而非将其误判为「尚未配置」。

## 常见任务

### 更换角色头像

打开「角色」页，找到目标角色卡，点击头像区域或「更换图片」按钮，弹出 MediaStudio 弹窗。

弹窗默认停在「AI 生成」标签页：
1. 在文本框内填写生成描述，越具体越好（例如「身着白色汉服的年轻女子，清澈眼神，水墨风格」）。
2. 在「图像生成模型」选择器中选好 provider 和模型（仅展示支持图像生成的模型）。
3. 在「尺寸 / 比例」中选择比例——角色头像默认推荐竖版 2:3（832x1216）。
4. 点击「生成」，后台异步处理，生成期间显示「生成中，请稍候…」。
5. 生成完成后自动关闭弹窗并更新头像。

你也可以切到「上传」标签页直接上传本地图片，或到「图库」标签页复用之前生成或上传过的图片。

### 更换剧本封面

打开「剧本」编辑页，点击封面区域，弹出 MediaStudio 弹窗。封面尺寸默认推荐宽屏 16:9（1344x768）。操作流程与更换角色头像相同。

### 生成 / 维护角色人设图

人设图是专属角色特征描述（外貌/性格/背景）的全身插图，与头像分开管理。

打开「角色」页，进入目标角色卡编辑，切换到「人设图」标签：

- **立即生成**：点击「立即生成」，系统根据当前角色的外貌、性格等描述自动组织提示词，加入异步队列，完成后自动更新人设图。
- **上传人设图**：点击「上传人设图」直接上传本地图片作为人设图。
- **自动同步**：开启「人设更新时自动同步人设图」开关后，每次保存角色卡时若人设内容发生变化，将自动触发重新生成。需已配置生图模型 API Key。
- **历史画廊**：「人设图历史」展示该角色历次生成记录，点击缩略图可将其设为当前人设图（同时更新头像）。

### 在游戏聊天中生成场景图

游戏或酒馆聊天框底部工具栏有「AI 生图」按钮（图标为图片样式）。点击后弹出生图弹窗：

1. 填写生成描述，例如描述当前场景氛围或角色动作。
2. 选择模型，选择尺寸（聊天内插图默认推荐方形 1:1）。
3. 点击「生成」。

生成完成后图片嵌入对应助手消息气泡内，不是单独一条图片消息。刷新页面后图片位置仍会保留（持久化到本地存储）。

### 选择分辨率 / 比例

所有生图弹窗中均有「尺寸 / 比例」选择器，共五档：

| 档位 | 比例 | 像素 | 适合场景 |
|------|------|------|---------|
| 竖版 2:3 | 2:3 | 832x1216 | 角色头像、人设图 |
| 方形 1:1 | 1:1 | 1024x1024 | 聊天插图、账户头像 |
| 横版 3:2 | 3:2 | 1216x832 | 通用横图 |
| 长竖 9:16 | 9:16 | 768x1344 | 手机竖屏风格 |
| 宽屏 16:9 | 16:9 | 1344x768 | 剧本封面 |

选过一次后平台会按场景（kind）记住你的偏好，下次打开同类弹窗自动恢复上次选的档位。注意：Vertex AI（Gemini 系列）暂不支持自定义尺寸，选了也会被忽略。

### 取消生成中的任务

右下角的「后台任务」浮窗会显示正在进行的生图任务。找到对应任务后点击「取消」按钮可中止。

### 裁剪图片

点击已上传或已生成的图片可全屏预览。预览界面点击「裁剪」进入裁剪模式：拖动裁剪框或拖动边角 handle 调整区域，点击「应用裁剪」确认。裁剪后自动压缩（最长边限 1280px，JPEG 0.85 质量）并上传。

## 常见问题

**提示「尚未配置生图模型 API Key」怎么办？**

进入「设置 → API 密钥」，为你要使用的生图服务商填写 API Key；再进入「设置 → 模块模型」，找到「图像生成模型」一行，选择对应 provider 和模型。两步都完成后重新点击「生成」即可。

**提示「今日生图次数已达上限」是什么意思？**

部分服务商账号存在每日调用量限制，与平台无关，上限由你自己的 API Key 所属账号决定。换一个 Key 或等次日重置。

**生图选择器里没有出现我配置的模型？**

模型选择器仅展示带有图像生成能力标记的模型。如果你配置的模型不在列表里，可能是该模型本身不支持生图，或 provider 配置里的 base_url 指向了纯文本端点。

**Vertex AI 选了尺寸但没效果？**

目前 Vertex AI（Gemini 系列）不支持通过参数自定义尺寸，尺寸选项会被后端忽略，图片按模型默认比例输出。

**人设图「立即生成」后看不到变化？**

人设图生成是异步队列，提交后需等待一段时间，完成后头像和人设图历史会自动更新。如果开了「人设更新时自动同步」，每次保存角色卡也会自动触发，无需手动点击。

**图片生成后显示在哪里？**

- 角色头像、剧本封面、人设图：生成完成后直接更新对应位置，并进入「文件库」归档。
- 聊天 / 游戏场景图：嵌入对应助手消息气泡内，刷新后仍保留。
- 历史图片：可在角色卡「人设图历史」或「文件库」查看。
